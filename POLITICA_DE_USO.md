# Política de Uso - SipSignal Bot

## 1. Aceptación de Términos

Al utilizar el bot SipSignal (en adelante "el Bot"), aceptas cumplir con estos términos y condiciones. Si no estás de acuerdo, por favor no utilices el Bot.

## 2. Naturaleza del Servicio

SipSignal es un bot de Telegram que proporciona:
- Información sobre precios de criptomonedas
- Análisis técnico automatizado
- Alertas de precios personalizables
- Herramientas de seguimiento de mercado

**IMPORTANTE:** La información proporcionada por el Bot es únicamente con fines informativos y educativos. NO constituye asesoramiento financiero, de inversión o comercial.

## 3. Descargo de Responsabilidad

### 3.1 Riesgo de Inversión
- Las criptomonedas son inversiones de alto riesgo
- Los precios pueden ser extremadamente volátiles
- Puedes perder todo tu capital invertido
- El Bot no garantiza la exactitud de los datos

### 3.2 Exactitud de Datos
- Los datos se obtienen de fuentes de terceros
- Pueden existir retrasos o errores en la información
- El Bot no se responsabiliza por decisiones tomadas basadas en sus alertas

### 3.3 Sin Garantías
EL BOT SE PROPORCIONA "TAL CUAL", SIN GARANTÍAS DE NINGÚN TIPO, EXPRESAS O IMPLÍCITAS.

## 4. Uso Aceptable

### 4.1 Prohibiciones
Está estrictamente prohibido:
- Usar el Bot para actividades ilegales
- Intentar acceder sin autorización a sistemas del Bot
- Sobrecargar o atacar el servicio (DoS/DDoS)
- Distribuir malware o contenido malicioso
- Suplantar la identidad de otros usuarios
- Extraer datos de usuarios sin consentimiento

### 4.2 Límites de Uso
- Se aplican límites de rate para prevenir abuso
- El uso excesivo puede resultar en bloqueo temporal
- Las suscripciones gratuitas tienen límites establecidos

## 5. Privacidad y Datos

### 5.1 Información Recopilada
- ID de usuario de Telegram
- Preferencias de configuración (monedas, alertas)
- Historial de uso básico

### 5.2 Uso de Datos
- Los datos se utilizan únicamente para operar el Bot
- No vendemos ni compartimos datos con terceros
- Los datos se almacenan de forma segura

### 5.3 Retención
- Puedes solicitar la eliminación de tus datos
- Los datos se conservan mientras uses el servicio
- Los logs se eliminan periódicamente

## 6. Suscripciones y Pagos

### 6.1 Servicios Gratuitos
- Acceso básico a funcionalidades
- Límites de uso aplicables
- Sin garantía de disponibilidad

### 6.2 Servicios Premium (VIP)
- Beneficios adicionales según el plan contratado
- Pagos procesados a través de plataformas seguras
- No hay reembolsos una vez activado el servicio

## 7. Terminación

Nos reservamos el derecho de:
- Suspender o terminar cuentas que violen estas políticas
- Modificar o discontinuar el servicio con previo aviso
- Bloquear usuarios que abusen del sistema

## 8. Modificaciones

Esta política puede ser actualizada periódicamente. Los cambios significativos serán notificados a los usuarios.

## 9. Contacto
